Podstawowy model - encja to użytkowni

Przygotuje widok listy użytwników która ma następując pola

id, uuid, login nazwa, status (active|disabled)

Chcę móc listować rekordy, oraz móc dodwać i edytować rekory 

Dane do encji user będą pobierane z API backenu o adresie http://127.0.0.1:3000

Dane będą generowane za pomocą narzedzia json server. 

Proszę zaproponuj pełną implementację podzieloną na trzy cześci. 

Lista userów, następnie dycja userów i potem dodawanie userów. Na każdyż z tych etapów będę chciał uruchomić kod i sprawdzić działanie.
